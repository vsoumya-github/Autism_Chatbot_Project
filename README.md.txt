# Autism Support Chatbot (AI Project)

This is a simple AI-based chatbot designed to provide basic support and information about autism. It is written in Python and built using Google Colab, and Gradio for user interface.

## Features:
- Responds to questions about autism, symptoms, diagnosis, treatment, sleep issues, etc.
- User-friendly interface using Gradio
- Educational purpose project for building a beginner AI portfolio

## Tools Used:
- Python
- Google Colab
- Gradio
- (Upcoming: testRigor, TensorFlow, etc.)

## How to Run:
Open the `.ipynb` notebook using Google Colab and run all cells. It will launch a chatbot UI.
